

## OneYardPass

## 系统主要模块页面设计

### 用户模块

1. 用户注册

   > 用户如需使用本系统进行跨境通关预约，需要在本系统通过真实手机号码并设置登录密码成功注册个人身份信息后，才可正常使用本系统进行跨境通关申报等操作。

   - 用户注册表单

     |  字段  |  输入类型   |             要求             |             备注             |
     | :--: | :-----: | :------------------------: | :------------------------: |
     |  密码  | 数字、字母组合 | 必填项，且以大写字母开头的不少于6位的字母、数字组合 |         不符合要求提示修改          |
     | 校验密码 | 数字、字母组合 | 必填项，且以大写字母开头的不少于6位的字母、数字组合 |    两次输入密码必须相同，不同将会提示修改     |
     | 手机号码 |   数字    |       必填项，11位可用手机号码        | 用于获取注册验证码，成功注册后将作为用户的唯一用户名 |
     | 验证码  |   数字    |            必填项             | 填写用户所填手机号码接收的验证码，验证通过即可注册  |

     补充说明： 

     - 用户可点击“立即注册”按钮提交注册信息，交由后台审核；
     - 表单必填项使用*符号标注，未填写则不可点击注册按钮提交注册信息；
     - 注册表单内容填写不符合要求或存在错误不可点击注册按钮提交注册信息，且弹出相关提示信息，提示用户修改。
     - 提交后根据后台返回信息提示用户是否注册成功，成功则自动跳转至登录界面提示用户登录；反之提示失败信息，提示用户重新注册。

     ​

2. 用户登录

   > 用户注册成功后即可使用注册所用手机号码以及密码通过登录界面登入本系统。

   - 用户登录表单

     |  字段  |  输入类型   |             要求             |     备注      |
     | :--: | :-----: | :------------------------: | :---------: |
     | 用户名  |   数字    |       必填项，11位可用手机号码        | 用户注册所使用手机号码 |
     |  密码  | 数字、字母组合 | 必填项，且以大写字母开头的不少于6位的字母、数字组合 |  用户注册所设置密码  |

     补充说明：

     - 用户使用本系统注册且正常登录后才可正常使用相关功能；
     - 表单必填项使用*符号标注，未填写则不可点击登录按钮提交登录信息；
     - 用户登录所用用户名为注册时所用手机号码，密码即为注册时所设置密码；
     - 登录表单内容填写不符合要求或存在错误不可点击登录按钮提交登录信息，且弹出相关提示信息，提示用户修改。
     - 提交后根据后台返回信息提示用户是否登录成功，成功则进入系统首页；反之提示失败信息，如“用户名不存在”、“账号密码不匹配”等，提示用户修改或前往注册。

     ​

3. 忘记/找回密码

   > 如用户忘记登陆密码，可通过找回密码操作重新设置用户登录密码。

   - 用户身份验证

     > 对用户身份进行验证

     | 字段   | 输入类型 | 要求            | 备注                            |
     | ---- | ---- | ------------- | ----------------------------- |
     | 手机号码 | 数字   | 必填项，11位可用手机号码 | 用户注册时所使用的手机号码，即用户名            |
     | 验证码  | 数字   | 必填项           | 填写用户所填手机号码接收的验证码，验证通过即可重新设置密码 |

     补充说明：

     - 对用户身份进行验证，验证通过即可重新设置用户密码；

     ​

   - 找回密码

     > 重新设置用户密码

     | 字段    | 输入类型    | 要求                         | 备注                  |
     | ----- | ------- | -------------------------- | ------------------- |
     | 新密码   | 数字、字母组合 | 必填项，且以大写字母开头的不少于6位的字母、数字组合 | 不可与旧密码相同            |
     | 校验新密码 | 数字、字母组合 | 必填项，且以大写字母开头的不少于6位的字母、数字组合 | 两次输入密码必须相同，不同将会提示修改 |

     补充说明：

     - 设置新密码不可与旧密码相同；
     - 重新设置密码请求提交并审核通过后，跳转登陆界面提示用户登陆。

     ​

4. 用户个人中心

   > 用户注册并成功登入后，可通过“个人中心”按钮进入用户个人中心，在此用户可设置自身相关信息，如设置“预约通关车辆牌照”、“修改用户密码”等操作。

   - 设置“预约通关车辆牌照”

     > 用户提交跨境通关申请时需选择通关时所驾驶车辆，用户需通过此模块设置相关信息后才可正常使用。

     |   字段    |     输入类型      |        要求        |  备注  |
     | :-----: | :-----------: | :--------------: | :--: |
     |  牌照号码   | 汉字、符号、数字、字母组合 | 必填项，注册在案的可用车辆牌照  |      |
     |  车主姓名   |      汉字       |  必填项，注册在案的合法姓名   |      |
     | 车主身份证号码 |    数字、字母组合    | 必填项，注册在案的合法身份证号码 |      |

     补充说明： 

     - 设置“预约通关车辆牌照”且通过审核后，用户可使用该车辆进行跨境通关申请；
     - 用户必须设置至少一个“预约通关车辆牌照”后才可进行跨境通过申请；
     - 表单必填项使用*符号标注，未填写则不可点击提交按钮提交信息；
     - 用户所提交信息需通过后台审核，审核通过后即可使用。

     ​

   - ”修改用户密码“

     > 用户可通过此模块修改已设置用户密码。

     | 字段    | 输入类型    | 要求                         | 备注                  |
     | ----- | ------- | -------------------------- | ------------------- |
     | 旧密码   | 数字、字母组合 | 必填项，且以大写字母开头的不少于6位的字母、数字组合 |                     |
     | 新密码   | 数字、字母组合 | 必填项，且以大写字母开头的不少于6位的字母、数字组合 | 不可与旧密码相同            |
     | 校验新密码 | 数字、字母组合 | 必填项，且以大写字母开头的不少于6位的字母、数字组合 | 两次输入密码必须相同，不同将会提示修改 |

     补充说明： 

     - 用户要修改密码需提供旧密码，以验证用户身份；
     - 设置新密码不可与旧密码相同；
     - 修改密码请求提交并审核通过后将移除本次登陆操作，跳转登陆界面提示用户身份信息过期需重新登陆。

     ​

     ​

### 跨境通关预约模块

1. 跨境通关预约

> 用户正常注册并登录后，当需要进行跨境通关时，需提前通过本系统提交相关信息进行预约，预约通过后可通过制定关口进行通关。

- | 字段                  | 输入类型       | 要求             | 备注                              |
  | ------------------- | ---------- | -------------- | ------------------------------- |
  | 口岸编号                | 汉字、字母组合    | 必填项，现行可用关口编号   |                                 |
  | 出入境类型               | 汉字         | 必填项            | 1.出境  2.入境                      |
  | 车主姓名                | 汉字         | 必填项，注册在案用户姓名   |                                 |
  | 联系电话                | 数字         | 必填项，可用手机号码     |                                 |
  | 车辆牌照                | 汉字、字母、数字组合 | 必填项，注册在案合法车辆牌照 |                                 |
  | 集装箱编号               | 字母、数字组合    | 必填项，注册在案集装箱编号  |                                 |
  | 货物类型                | 汉字         | 必填项，关口所允许运载货物  | 1.医药类  2.易燃物  3.食品  4.五金类  5.其他 |
  | 货物重量                | 数字         | 必填项，单位吨（t）     |                                 |
  | 开始时间（最早可通关时间段起始时间点） | 数字         | 必填项            | 示例：2021-09-09-15:00             |
  | 结束时间（最晚可通关时间段结束时间点） | 数字         | 必填项            | 示例：2021-09-09-17:00             |

  补充说明

  - 用户需提交申报表并在审核通过后获取通关所需二维码才可通关；
  - 必填项为必须正确填写后即可提交，否则不可提交申请；
  - 货物类型根据后天返回结果提供给用户选取；
  - 货物重量根据用户所选货物类型从后台获取相应重量限制，以供用户选择；
  - 开始时间和结束时间，用户所选通关时间段的最早时间后最迟时间，用户在此时间段内可正常通关，不在指定时间段则不可通关；
  - 口岸编号由后台返回数据将可选口岸展示给用户以选择，并在用户选择出入境类型后自行展示起始和目的地。

  ​

1. 预约查询

   > 用户可在“预约查询”页面查看已提交的所有预约申请、查看预约的审核进度、查看已通过预约的通关二维码、修改被退回的不合规预约申请、对正在审核过程中的预约信息进行修改、撤回已提交的预约申请等操作。

   - 所有预约列表展示页

     > 展示所有以提交的预约申报表

     | 字段   | 输入类型    | 要求   | 备注                  |
     | ---- | ------- | ---- | ------------------- |
     | 口岸编号 | 汉字、字母组合 |      | 某站1号口               |
     | 起始地  | 汉字      |      | 示例：缅甸               |
     | 目的地  | 汉字      |      | 示例：中国               |
     | 发起时间 | 数字      |      | 示例：2021-09-09-15:00 |
     | 审核状态 | 汉字      |      | 示例：审核中/审核失败/审核通过    |

     补充说明：

     - 用户在可展示页查看所有已提交申请的简略信息，如口岸编号、起始点、目的地、发起时间、审核状态等；
     - 起始地与目的地以”缅甸 -中国“的形式向用户展示；
     - 不同审核状态使用不同字体颜色标注：审核中—橙色、审核失败—红色、审核通过—绿色；
     - **用户可手动选择筛选展示指定状态的申报表，如审核中或审核失败的申报表。**

     ​

   - 预约详情页

     > 用户可点击展示页进入详情页查看更详细的申报信息

     | 字段                  | 输入类型       | 要求   | 备注                              |
     | ------------------- | ---------- | ---- | ------------------------------- |
     | 口岸编号                | 汉字、字母组合    |      |                                 |
     | 出入境类型               | 汉字         |      | 1.出境  2.入境                      |
     | 车主姓名                | 汉字         |      |                                 |
     | 联系电话                | 数字         |      |                                 |
     | 车辆牌照                | 汉字、字母、数字组合 |      |                                 |
     | 集装箱编号               | 字母、数字组合    |      |                                 |
     | 货物类型                | 汉字         |      | 1.医药类  2.易燃物  3.食品  4.五金类  5.其他 |
     | 货物重量                | 数字         |      |                                 |
     | 开始时间（最早可通关时间段起始时间点） | 数字         |      | 示例：2021-09-09-15:00             |
     | 结束时间（最晚可通关时间段结束时间点） | 数字         |      | 示例：2021-09-09-17:00             |
     | 通关二维码（审核通过）         |            |      | 审核已通过的预约会展示通关所需的二维码，以供用户通关使用    |
     | 撤销（审核中）             |            |      | 审核中的预约用户可进行撤回操作已取消预约或修改信息       |
     | 修改（审核失败）            |            |      | 审核失败的申报表会被退回，用户可进行修改后重新提交       |

     补充说明：

     - 详情页展示详细申报信息；
     - 已审核通过的申报表不可修改，并展示通关所需二维码，以供用户通关时使用；
     - 审核中的申报表用户可进行撤销操作以撤销预约或修改预约信息；

### 管理员模块

1. 管理员用户登入

   > 管理员用户通过后台直接添加后可正常通过登录功能登入本系统，本系统会根据管理员相关权限展示不同页面以供管理员对普通用户的预约信息进行审核操作。

2. 管理员用户权限

   > 管理员权限由后台直接设置，本系统会根据后台返回相关权限信息展示不同界面以供管理员对普通用户的预约申请进行审核。

3. 管理员审核界面—列表页

   | 字段   | 输入类型    | 要求   | 备注                  |
   | ---- | ------- | ---- | ------------------- |
   | 口岸编号 | 汉字、字母组合 |      | 某站1号口               |
   | 起始地  | 汉字      |      | 示例：缅甸               |
   | 目的地  | 汉字      |      | 示例：中国               |
   | 发起时间 | 数字      |      | 示例：2021-09-09-15:00 |

   补充说明：

   - 展示所用该管理员所需要审核的普通用户提交的申报表，管理员可进入某一申报表查看用户所填写预约信息并进行审核。

   ​

4. 管理员审核界面—详情页

   | 字段                  | 输入类型       | 要求   | 备注                                  |
   | ------------------- | ---------- | ---- | ----------------------------------- |
   | 口岸编号                | 汉字、字母组合    |      |                                     |
   | 出入境类型               | 汉字         |      | 1.出境  2.入境                          |
   | 车主姓名                | 汉字         |      |                                     |
   | 联系电话                | 数字         |      |                                     |
   | 车辆牌照                | 汉字、字母、数字组合 |      |                                     |
   | 集装箱编号               | 字母、数字组合    |      |                                     |
   | 货物类型                | 汉字         |      | 1.医药类  2.易燃物  3.食品  4.五金类  5.其他     |
   | 货物重量                | 数字         |      |                                     |
   | 开始时间（最早可通关时间段起始时间点） | 数字         |      | 示例：2021-09-09-15:00                 |
   | 结束时间（最晚可通关时间段结束时间点） | 数字         |      | 示例：2021-09-09-17:00                 |
   | 通过（审核通过）            |            |      | 审核信息无误，通过该预约申请                      |
   | 退回（审核失败）            |            |      | 审核信息有误，驳回该预约申请                      |
   | 退回原因（审核失败原因）        |            |      | 审核失败的原因，由管理员填写返回给用户查看，提示用户修改后重新提交审核 |

   补充说明：

   - 管理员对普通用户的预约申请进行审核，选择是否通过预约申请；
   - 如用户预约申报表审核未通过，管理员需填写不通过原由，以提示用户。
